## Installation

### Steps

1. Clone the repository
    ```bash
    git clone https://github.com/Siratul804/Siratul12
    ```
2. Navigate to the project directory
    ```bash
    cd your-repo
    ```
3. Install dependencies
    ```bash
    npm install
    ```
4. Run the development server
    ```bash
    npm run dev
    ```
